# Proyecto DAM

Este es nuestro proyecto de Desarrollo de Aplicaciones Multiplataforma.

## Funcionalidades
- Conexión con API RAWG
- Integración con YouTube
- Repositorio de videojuegos 
- Conexion de usuarios
- Biblioteca para guardar favoritos
- Busqueda de videojuegos
